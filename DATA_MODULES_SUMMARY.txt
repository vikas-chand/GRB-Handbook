================================================================================
MODULE DESCRIPTIONS
================================================================================

1. DOWNLOADERS.PY
   Purpose: Core file downloading and caching utility
   Features:
     - Exponential backoff retry mechanism (configurable)
     - tqdm progress bars for all downloads
     - URL-based cache with MD5 hashing
     - Age-based cache cleanup
     - FITS file download support
   Key Classes: DataDownloader

2. SWIFT_FETCHER.PY
   Purpose: Swift X-ray Observatory data access
   Instruments Supported:
     - BAT (Burst Alert Telescope)
     - XRT (X-Ray Telescope)
     - UVOT (UV/Optical Telescope)
   Features:
     - Catalog queries (GRB database)
     - Light curve downloads
     - Spectral data access
     - Trigger ID resolution
   Key Classes: SwiftDataFetcher, LightCurveData, SpectralData

3. FERMI_FETCHER.PY
   Purpose: Fermi Gamma-ray Space Telescope data access
   Instruments Supported:
     - GBM (Gamma-ray Burst Monitor)
     - LAT (Large Area Telescope)
   Features:
     - Catalog queries
     - Light curve downloads (TTE and CSPEC formats)
     - Multi-detector support (12 NaI + 2 BGO detectors)
     - Spectral analysis
   Key Classes: FermiDataFetcher, LightCurveData, SpectralData

4. GCN_PARSER.PY
   Purpose: GCN Circular and Notice parsing
   Data Sources:
     - GCN3 Circulars (text archives)
     - Automated GCN Notices (machine-readable)
   Features:
     - Circular downloads and archive searching
     - Metadata extraction (title, authors, date)
     - Coordinate parsing (decimal degrees & HMS)
     - Scientific information extraction:
       * Redshift values
       * T90 (burst duration)
       * Flux density measurements
   Key Classes: GCNParser, GCNCircular, FluxMeasurement

5. MULTIMESSENGER.PY
   Purpose: Multi-messenger source cross-matching
   Supported Messengers:
     - Gravitational Waves (GWOSC)
     - Neutrinos (IceCube)
     - Astronomer's Telegrams (ATel)
   Features:
     - Event database queries
     - Temporal coincidence detection
     - Angular separation calculations
     - Significance scoring
   Key Classes: MultiMessengerQuery, GRBEvent

Multi-Messenger Analysis:
```python
from grb_pipeline.data import MultiMessengerQuery, GRBEvent

mmq = MultiMessengerQuery(config)
grb = GRBEvent(name='GRB 230101A', ra=123.45, dec=-45.67, trigger_time=1234567890)
report = mmq.check_multimessenger_associations(grb)
```

GCN Information Extraction:
```python
from grb_pipeline.data import GCNParser

parser = GCNParser(config, downloader)
circulars = parser.fetch_circulars_for_grb('GRB 230101A')
for text in circulars:
    parsed = parser.parse_circular_basic(text)
    print(f"Redshift: {parsed.redshift}, T90: {parsed.t90}")
```
